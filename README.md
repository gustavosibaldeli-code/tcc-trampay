# Sistema Trampay

Sistema web estilo marketplace de serviços.

## Tecnologias
- PHP
- MySQL
- HTML
- CSS

## Funcionalidades
- Cadastro de clientes e profissionais
- Login
- Sistema de agendamento
- Pagamentos

## Como rodar
1. Instalar XAMPP
2. Colocar na pasta htdocs
3. Importar banco de dados
4. Acessar localhost
